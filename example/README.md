# Smart Search List Examples

Interactive demos for every major feature of the `smart_search_list` package.

## Running

```bash
cd example
flutter run
```

## Demos

| Demo | What it shows |
|------|---------------|
| **Getting Started** | Minimal setup -- items, searchableFields, itemBuilder |
| **Basic Offline List** | Configuration options, pull-to-refresh |
| **E-commerce Products** | External controller with filters and sorting |
| **Fuzzy Search** | Typo-tolerant matching with SearchHighlightText |
| **Async API Loading** | Paginated network loading with error/retry |
| **Multi-Select** | Checkbox selection with select all/deselect all |
| **Grouped List** | Items grouped by category with headers |
| **Sliver Integration** | SliverSmartSearchList in a CustomScrollView |
| **Grouped Sliver** | Sticky group headers with SliverMainAxisGroup |
| **Empty States** | "No data" vs "no search results" comparison |
| **Search Trigger Modes** | onEdit (debounced) vs onSubmit (manual) |
| **Advanced Configuration** | External controller, custom builders, dynamic config |
| **Performance Test** | 10,000 items at 60 FPS |
| **Accessibility** | Localized labels, screen reader announcements |
